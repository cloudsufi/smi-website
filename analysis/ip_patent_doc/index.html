IP Landscape Overview

The millwork estimating software market has a dramatically uneven IP landscape. Legacy players (Alliance Millsoft, Cabinet Vision, INNERGY) have built moats through customer lock-in, proprietary databases, and switching costs — not patents. The new wave of AI-driven entrants (Togal.AI, and now MillworkSuite) is where active patent filing is occurring. Bluebeam, as the largest platform, holds an extensive legacy patent portfolio on PDF annotation and collaboration technology — but has minimal coverage on AI-driven scope extraction or millwork-specific workflows.

Key Finding
The patent space for AI-based millwork-specific scope extraction from architectural drawings is essentially unoccupied. Togal.AI has patented general AI construction takeoff from floor plans. Bluebeam has patented PDF tools. Nobody has patented the millwork-specific element detection and CAD library matching workflow that MillworkSuite is building. This is the most valuable IP opportunity in the market right now.

MillworkSuite's Patentable Innovations

Based on the publicly disclosed feature set at cloudsufi.github.io/smi-website/millworksuite, MillworkSuite has six distinct patent opportunities. Urgency is rated based on competitive proximity, novelty, and commercial value.

MSW-P1 · Priority: CRITICAL
AI-Based Millwork Element Detection and Classification from Architectural Drawings

Method for automatically detecting and classifying millwork-specific elements (base cabinets, upper cabinets, tall cabinets, vanities, countertops, casework units) from architectural PDF drawings using AI/ML, as distinct from general room area or floor space detection.

Why this is novel: Togal's patents cover general area/room detection. Bluebeam covers symbol matching. No patent covers millwork-type classification from architectural drawings specifically.

Key claims to draft: The classification taxonomy (base/upper/tall/vanity/countertop/casework); the multi-sheet detection spanning floor plans AND elevations; the confidence scoring for millwork type assignment.

High Novelty File Immediately USPTO + PCT
MSW-P2 · Priority: CRITICAL
System for Automated Elevation Drawing — Floor Plan Callout Linking

System and method for automatically detecting elevation callout markers in architectural floor plans and dynamically linking them to their corresponding elevation detail sheets within a multi-page PDF drawing set, enabling synchronized cross-sheet navigation.

Why this is novel: Bluebeam's hyperlink tool is manual. No competitor has automated this cross-referencing. This is genuinely new: extracting the callout symbol, parsing the reference code, and matching it to the correct sheet automatically.

Key claims to draft: The marker detection algorithm; the reference code parsing (letter/number callout conventions); the cross-sheet linking; the synchronized split-view display triggered by the link.

Very High Novelty File Immediately USPTO + PCT
MSW-P3 · Priority: HIGH
Automated Page Classification System for Architectural Millwork Drawing Sets

Method for automatic classification, naming, and organization of pages within an architectural PDF drawing package by drawing type (floor plan, elevation, section, detail, schedule, specification) using AI, with particular training for millwork-relevant sheet types and naming conventions.

Why this is novel: Togal auto-names sheets generically. No tool auto-classifies by millwork-relevant drawing type (casework elevations vs. architectural elevations, millwork schedules vs. door schedules). The training dataset distinction is key.

High Novelty File Within 6 Months USPTO Priority
MSW-P4 · Priority: HIGH
Method for CAD Design Library Matching from AI-Detected Millwork Scope

System and method for automatically matching AI-detected millwork elements from architectural drawings to corresponding product entries in CAD design software libraries (e.g., Microvellum, Cabinet Vision) based on detected dimensions, millwork type, and configuration parameters — enabling direct export with library pre-population.

Why this is novel: This is entirely unprecedented. No competitor product — not Bluebeam, not Togal, not any estimating tool — automatically matches detected scope from drawings to a user's CAD library. The "library pre-matching" described on the MillworkSuite website is a genuinely novel system.

Very High Novelty File Within 6 Months USPTO + PCT
MSW-P5 · Priority: MEDIUM
Synchronized Dual-Pane Viewer for Floor Plan and Elevation Navigation

Apparatus for displaying an architectural floor plan and its corresponding elevation drawing simultaneously in synchronized panes, where interaction with a floor plan callout marker dynamically loads the matching elevation in the adjacent pane using the cross-sheet link data from MSW-P2.

Why this is novel: While split-view PDF viewers exist, the automatic elevation-to-floor-plan synchronization triggered by callout markers is a novel interface method specific to architectural drawing navigation.

Medium Novelty File Within 12 Months USPTO Priority
MSW-P6 · Priority: MEDIUM
Structured Millwork Scope Export with Pre-Built Pricing Formula Templates

Method for generating a structured millwork scope of work document from AI-detected elements, with automatically populated pricing formula placeholders calibrated to detected millwork categories, dimensions, and quantity estimates — enabling direct use in cost estimation without manual re-entry.

Why this is novel: Distinguishable from generic "export to Excel" by the scope-categorization and formula-structure layer. The pricing formula schema tied to millwork element classification is the novel component.

Medium Novelty File Within 12 Months USPTO Priority

The Full IP Protection Shield — Beyond Patents

Patents are one layer of protection. A comprehensive IP strategy for MillworkSuite uses five layers, each protecting different aspects of the business against different threat vectors.

Layer 1 — Patents
What to protect: The six AI/workflow inventions identified (MSW-P1 through P6). Focus on method claims (how the AI detection works) and system claims (the architecture that enables library matching). Provisional applications can be filed in days and secure a priority date for 12 months while full applications are developed.

Timeline: File provisionals for MSW-P1 and MSW-P2 immediately. Provisionals for P3/P4 within 6 months. Full PCT application within 12 months for international protection.

Cost estimate: Provisional: $3,000–$8,000/application (with attorney). Full utility: $10,000–$25,000/application. PCT international: $5,000–$15,000 additional per jurisdiction.

Key risk: Software patents face § 101 eligibility challenges (Alice Corp. ruling). Claims must be drafted to cover a technical improvement to computing, not just an abstract idea. The AI training on millwork-specific data and the CAD library matching workflow are strong candidates for technical improvement framing.
Layer 2 — Trade Secrets
What to protect as trade secrets:
  • The AI training dataset and labeling methodology for millwork element detection
  • The confidence scoring model for millwork-type classification
  • The matching algorithm for CAD library mapping (specific rules, weighting, fallbacks)
  • The callout marker parsing rules and drawing convention database
  • Customer usage data and pricing calibration data
Protection mechanisms: NDAs with all employees, contractors, and partners. Restricted access controls. No publication of implementation details. Source code is not disclosed in patent applications (only method claims).

Why this matters: Trade secrets protect what patents cannot (implementation details) and have no expiration date as long as secrecy is maintained.
Layer 3 — Copyright
Automatic protection, no filing required: All source code, UI designs, documentation, and training materials are automatically copyright protected from creation. However, register key works with the Copyright Office to enable statutory damages in infringement suits.

Register immediately:
  • The MillworkSuite software application (source code registration)
  • The UI/UX design and visual elements
  • Training datasets and labeled image data (to the extent they constitute original creative work)
  • All user-facing documentation and onboarding materials
Cost: $65–$85/application online via Copyright.gov. Registration enables statutory damages up to $150,000/work for willful infringement.
Layer 4 — Trademarks
File immediately for:
  • MILLWORKSUITE (word mark) — Class 42 (software as a service)
  • MILLWORKSUITE + Logo (design mark) — Class 42
  • Any distinctive product feature names (e.g., "Floor Plan Intelligence," "AI Elevation Detection" — assess distinctiveness)
Why urgency: Trademark protection is first-use-based in the US, but federal registration provides critical enforcement tools, nationwide notice, and the basis for blocking infringing domain names. File before public launch — use-in-commerce application or intent-to-use (ITU) application if not yet commercially active.

Cost: ~$350/class/mark for USPTO filing. Attorney fees: $1,000–$3,000 for a full trademark clearance search and filing.
Layer 5 — Contracts & Access Controls
Key contractual IP protections:
  • Employee IP Assignment Agreements: All employees and contractors must sign IP assignment agreements before contributing to MillworkSuite — ensures all inventions are owned by the company
  • NDAs: With all partners, prospects, pilot customers, and integrators (Microvellum, Cabinet Vision resellers)
  • End User License Agreement (EULA): Prohibit reverse engineering, competitive benchmarking disclosure, and data scraping
  • Terms of Service: Address customer data ownership, AI training use of uploaded drawings, and confidentiality
  • Partner Agreements: Explicit IP ownership clauses in any co-development or integration agreements with Microvellum or Hexagon

Final Recommendation
MillworkSuite's most durable competitive moat will be built from three sources: (1) patents on the millwork-specific AI detection and CAD library matching workflows — file now; (2) trade secrets on the training data, model architecture, and matching rules — protect rigorously; and (3) network effects through deep integrations with Microvellum and Cabinet Vision that make MillworkSuite progressively more valuable as customer libraries are built in and switching costs grow. The patent portfolio protects the technology. The trade secrets protect the intelligence. The integrations protect the customer relationships.
